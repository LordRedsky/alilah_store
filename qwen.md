🔑 Struktur Landing Page
1. 	Header
• 	Logo atau nama brand Alilah Store
• 	Navigasi sederhana (scroll ke bagian produk)
2. 	Hero Section
• 	Judul singkat yang menjelaskan produk/brand
• 	CTA utama: “Lihat Produk”
3. 	Produk Section (3 produk)
• 	Card per produk:
• 	Foto produk
• 	Nama & deskripsi singkat
• 	Harga
• 	Tombol “Pesan via WhatsApp”
• 	Tombol ini akan membuka link 
4. 	Testimoni / Keunggulan
• 	Bisa ditambahkan untuk meningkatkan kepercayaan
5. 	Footer
• 	Informasi kontak
• 	Link sosial media

📱 Responsif Mobile & Tablet
• 	Gunakan flexbox / grid untuk card produk.
• 	Pada mobile → card ditampilkan vertikal (satu per baris).
• 	Pada tablet → card bisa 2 kolom.
• 	Pastikan tombol WhatsApp besar dan mudah diklik.
